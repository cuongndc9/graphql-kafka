# graphql-kafka

Integrating 💩 Kafka with GraphQL.

## How to start the application?

### Run Kafka & ZooKeeper containers

```shell
docker-compose up -d
```

### Run the application

```shell
yarn install
yarn start:dev
```

### Go to http://0.0.0.0:5000 & enjoy it
